# AverlistUI

## Установка

Установка используя yarn:

```
yarn add averlist-ui
```

Установка используя npm:

```
npm i averlist-ui
```

## Пример

### Импорт

#### CommonJS

```ts
const { Button } = require('averlist-ui');
```

#### ES6

```ts
import { Button } from 'averlist-ui';
```

### Использование

```tsx
import { Loader } from 'averlist-ui'


export const Component = () => {
	return (
		<Loader size={20} />
    )
}
```
